# 子集系统

在了解拟阵前，需要先了解子集系统。子集系统定义在有序二元组M=(S,L)M=(S,L)M=(S,L)上，满足下面几个条件

  • 基础集合S是一个有限集。
  • L是由S子集构成的集合（注意L是集合的集合）且L非空。
  • 遗传性：∀A⊆B\forall A\subseteq B∀A⊆B，如果B⊆LB\subseteq LB⊆L，则A⊆LA\subseteq LA⊆L（即L中元素的子集必在L中）。

注意由第二、三点可知，空集∅\emptyset∅必然在L中（包含空集的集合不是空集🤣）。

# 独立集

独立集是拟阵中的元素，仍然用M=(S,L)M=(S,L)M=(S,L)表示子集系统。对于A⊆SA\subseteq SA⊆S，如果A∈LA\in LA∈L，则AAA为独立集。对于独立集AAA，若存在x∈Sx\in Sx∈S，满足x∉Ax\notin Ax∉A且A∪{x}∈LA\cup \{x\}\in LA∪{x}∈L，则称AAA为可扩展的。不可扩展的独立集为极大独立集。

它有几个性质：

  • 空集一定是独立的，也就是说，∅∈L\emptyset\in L∅∈L（与子集系统定义的遗传性的推论相符）。
  • 遗传性：每个独立集的子集是独立的（与子集系统定义遗传性相符）。
  • 交换性：如果AAA和BBB是LLL的两个独立集，AAA比BBB有更多的元素，则在AAA中存在一个元素，当其加入BBB时得到一个比BBB更大独立集 （与拟阵定义中的交换性相符）。

  3. 设{x}\{x\}{x}是对于加权拟阵M=(S,L)M=(S,L)M=(S,L)由定理2中所选择的S的第一个元素，则剩下问题可归结为求加权矩阵M′=(S′,L′)M'=(S',L')M​′​​=(S​′​​,L​′​​)的最大权独立集问题，其中S′={y∣y∈S,(x,y)∈L}S'=\{ y|y \in S, (x,y) \in L \}S​′​​={y∣y∈S,(x,y)∈L}，L′={B∣B∈S∖{x},B∪{x}∈L}L'=\{B|B\in S\setminus\{x\}, B\cup\{x\}\in L\}L​′​​={B∣B∈S∖{x},B∪{x}∈L}

# 案例分析：最小生成树(MST)

一个有 nnn 个结点的连通图的生成树是原图的极小连通子图，且包含原图中的所有$ n$ 个结点，并且有保持图连通的最少的边。即在一给定的无向图G=(V,E)G = (V, E)G=(V,E) 中，(u,v)(u, v)(u,v) 代表连接顶点 $u $与顶点 vvv 的边，而 w(u,v)w(u, v)w(u,v) 代表此边的权重，若存在 TTT 为 EEE 的子集且为无循环图，使得 w(T)w(T)w(T) 最小，则此 TTT 为 GGG 的最小生成树。

对于最小生成树问题，考虑无向图G=(V,E)G=(V,E)G=(V,E)，我们可以定义M=(S,L)M=(S,L)M=(S,L)：

# 补充：拟阵的秩

极大独立集的基数被定义为它所包含的元素个数。

对于拟阵M=(S,L)M=(S,L)M=(S,L)，LLL中的极大独立集称为拟阵的基。

拟阵的秩就是基的大小。

子集AAA的秩通过秩函数r(A)r(A)r(A)定义，它有以下几种性质:

  • 秩函数的值总是非负的
  • 对于任意EEE的子集AAA，有r(A)≤∣A∣r(A) \leq|A|r(A)≤∣A∣
  • 对于EEE的任意两个子集AAA 和BBB ，有r(A∪B)+r(A∩B)≤r(A)+r(B)r(A \cup B)+r(A \cap B) \leq r(A)+r(B)r(A∪B)+r(A∩B)≤r(A)+r(B)，这意味着秩函数是一个子模函数（见补充：子模函数）

# 补充：次模函数

次模（submodular）函数：又称“子模函数”或“亚模函数”，次模函数具有次模性（submodularity），它是经济学上边际效益递减（property of diminishing returns）现象的形式化描述（若对于连续函数，单调递增时，二阶导数递减）。
